Target Toggle

Show/hide elements by clicking a button. Supports fade animations, custom toggle classes, and button active states.

Basic Toggle (Inline Style)

Click the button to toggle the visibility of the login box. Uses inline style="display: none"

Login Form
<button ujs-target-id="#loginBox1">Toggle Login</button>
<div id="loginBox1" style="display: none;">
    Login form content...
</div>

Toggle with Class

Toggle using a CSS class instead of inline styles. More flexible for complex styling.

Menu
  • Dashboard
  • Profile
  • Settings
  • Logout
<button ujs-target-id="#menuBox" ujs-toggle-class="hide">Toggle Menu</button>
<div id="menuBox" class="hide">
    Menu content...
</div>

<style>
.hide { display: none; }
</style>

Custom Animation Duration

Control the speed of the fade animation with ujs-anim-duration

Slow fade: Takes 1 second to fade in/out
Fast fade: Takes only 150ms to fade in/out 
<button ujs-target-id="#slowFadeBox" ujs-anim-fade ujs-anim-duration="1000">
    Slow Fade
</button>

<button ujs-target-id="#fastFadeBox" ujs-anim-fade ujs-anim-duration="150">
    Fast Fade
</button>

Password Show/Hide Toggle

Automatically add a show/hide button to password inputs. Toggle between password and text type with a single attribute.

Basic Password Toggle

Simply add ujs-pass-show-hide attribute to any password input

<input type="password" 
       class="form-control" 
       ujs-pass-show-hide 
       placeholder="Enter your password" />

Custom Button Styling

Customize the toggle button appearance with ujs-pass-btn-class

<input type="password" 
       class="form-control" 
       ujs-pass-show-hide 
       ujs-pass-btn-class="text-primary fs-5"
       placeholder="Enter your password" />

Multiple Password Fields

Works with multiple password inputs on the same page

<div class="mb-3">
    <label class="form-label">Current Password</label>
    <input type="password" class="form-control" ujs-pass-show-hide />
</div>

<div class="mb-3">
    <label class="form-label">New Password</label>
    <input type="password" class="form-control" ujs-pass-show-hide />
</div>

Show More / Show Less

Collapse long content to a fixed height with a fade-out hint, then expand it fully with a button click. Works with any content — paragraphs, lists, mixed elements.

Instant Toggle (Hard Show)

No ujs-anim-fade attribute on the button means the expand/collapse snaps instantly.

CodeIgniter 3 is a lightweight PHP framework built for speed and simplicity. It follows the MVC pattern, has a small footprint, and requires minimal configuration to get started.

It ships with libraries for form validation, sessions, file uploads, database abstraction, and more, without forcing a lot of boilerplate on the developer.

While CI3 is in maintenance mode, it's still widely used in production apps due to its simplicity and predictable behavior.

This paragraph and anything below it is only visible once the content is expanded.

<div id="showMoreBox1" ujs-show-more-content ujs-show-more-height="100">
    <p>Paragraph 1...</p>
    <p>Paragraph 2...</p>
    <p>Paragraph 3...</p>
</div>
<button ujs-show-more-target="#showMoreBox1">Show More</button>

Add ujs-anim-fade (and optionally ujs-anim-duration) to the button for a smooth height transition.

  • Second item in the list
  • Third item, still visible
  • Fourth item, hidden until expanded
  • Fifth item, hidden until expanded
  • Sixth item, hidden until expanded
<div id="showMoreBox2" ujs-show-more-content ujs-show-more-height="80">
    <ul>
        <li>Item 1</li>
        ...
    </ul>
</div>
<button 
    ujs-show-more-target="#showMoreBox2" 
    ujs-anim-fade 
    ujs-anim-duration="400">
    Show More
</button>

Custom Labels & Fade Color

Use ujs-show-more-text / ujs-show-more-text-less for custom labels, and ujs-show-more-fade-color if your box isn't on a white background.

This box sits on a light grey background instead of white.

The fade overlay color attribute makes sure the gradient blends with the surrounding background instead of showing a white box.

Extra content that appears once expanded goes here, demonstrating the custom fade-out color matches the demo area.

<div 
    id="showMoreBox3" 
    ujs-show-more-content 
    ujs-show-more-height="90" 
    ujs-show-more-fade-color="#f1f3f5">

    ujs-show-more-target="#showMoreBox3" 
    ujs-anim-fade
    ujs-show-more-text="Read More"

Copy to Clipboard

Copy text from any element — visible inputs, hidden inputs, or plain elements like divs — to the clipboard with one click.

Copy From a Visible Input

Point ujs-copy-target at any input, textarea, or element.

<input type="text" id="shareLink1" class="form-control" value="https://example.com/invite/ab12cd34" readonly>
<button ujs-copy-target="#shareLink1">Copy</button>

Copy From a Hidden Input

Works even when the source input is a type="hidden" field, not just visible ones.

<input type="hidden" id="apiKey1" value="sk_live_9f8a7b6c5d4e3f2g1h0i">
<button 
    ujs-copy-target="#apiKey1" 
    ujs-copy-success-text="Key copied!">
    Copy API Key
</button>

Copy Plain Text From a Div

Copies the element's text content only (no HTML markup).

npm install utils-js-components 
<div id="codeBlock1">npm install utils-js-components</div>
<button 
    ujs-copy-target="#codeBlock1" 
    ujs-copy-success-duration="1500">
    Copy Command
</button>

Copy a Literal Value

No source element needed — use ujs-copy-value directly on the button.

<button ujs-copy-value="support@example.com">Copy Support Email</button>
